# Age-Calculator Version 1.0
A simple Age Calculator using HTML, CSS, JS
# What this project does
Takes the date entered by the user
Restricts future dates
Subtracts birth date from current date
Shows the age in terms of years, months and date
# Languages used
HTML
CSS
JS
# What I learned
Use of .value property to get the current value of input field
Use of .innerHTML property to edit/update the content of HTML element
Connecting JS to buttons in HTML using "onclick()" which is alternative to Event Listener
Use of date objects: new Date(), .getFullYear(), .getMonth(), .getDate(), .toISOString()
The use of string method : .split()
Restricting future dates by using Input Validation using .max
Using if-else condtions for borrowing months and days
